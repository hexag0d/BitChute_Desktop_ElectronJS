To get started:
1.) clone the repo
2.) open command prompt and cd to your repo root then run the following commands:
3.) npm install --save-dev electron
4.) npm install fluent-ffmpeg@2.1.2
5.) npm install jquery --save
6.) npm i custom-electron-titlebar
7.) npm start
